## API Reference

### Root

Groups the `Header` and `Body` parts. This component is based on the `table` element and supports common marginÂ props.

| Prop | Type | Default |
| --- | --- | --- |
| size | Responsive<"1" | "2" | "3"> | "2" |
| variant | "surface" | "ghost" | "ghost" |
| layout | Responsive<"auto" | "fixed"> |  |


### Header

Contains the column headings for the table, based on the `thead` element.

### Body

Displays the table data. This component is based on the `tbody` element.

### Row

A row of table cells. Based on the `tr` element.

| Prop | Type | Default |
| --- | --- | --- |
| align | Responsive<"start" | "center" | "end" | "baseline"> |  |
