## API Reference

This component is based on the `div` element and supports common margin props.

Only the `display` prop values are unique to the Box component.

| Prop | Type | Default |
| --- | --- | --- |
| as | "div" | "span" | "div" |
| asChild | boolean |  |
| display | Responsive<"none" | "inline" | "inline-block" | "block"> |  |
