## API Reference

This component inherits props and functionality from the Toggle Group primitive. It supports common marginÂ props.

### Root

Contains the items of the control.

| Prop | Type | Default |
| --- | --- | --- |
| size | Responsive<"1" | "2" | "3"> | "2" |
| variant | "surface" | "classic" | "surface" |
| radius | "none" | "small" | "medium" | "large" | "full" |  |


### Item

Represents individual values of the control.
